# Phonebook Backend - Full Stack Open 2026

This is the backend for the Phonebook application, developed as part of the **Full Stack Open** course from the University of Helsinki.

## 🚀 Live Demo
You can access the deployed application here:
[https://phonebook-tjmt.onrender.com/](https://phonebook-tjmt.onrender.com/)

## 🛠️ Features
- **Full Stack Integration**: Serves the production build of the React frontend.
- **RESTful API**: Supports fetching all persons, individual person details, and deleting entries.
- **Middlewares**: Uses `morgan` for logging, `cors` for cross-origin requests, and `express.static` for serving frontend files.
- **Deployment**: Automated build and deployment pipeline for Render.

## 📂 Project Structure
- `index.js`: The main entry point for the Express server.
- `dist/`: Contains the production build of the frontend (React).
- `package.json`: Script configurations for development and deployment.

## 💻 Tech Stack
- **Backend**: Node.js, Express.js
- **Frontend**: React (Vite), Tailwind CSS
- **Tools**: Morgan, CORS, Nodemon
